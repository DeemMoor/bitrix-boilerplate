# Проектный скил

Используй этот скил вместе с корневым `AGENTS.md`. `AGENTS.md` является основным источником проектных правил, а этот скил подключает их как локальный Codex skill.

## Основные правила

- Всегда пиши и отвечай пользователю на русском языке, если пользователь явно не попросил другой язык.
- Перед работой учитывай правила из `AGENTS.md`.
- Держи изменения строго в рамках поставленной задачи.
- Сохраняй существующую PHP- и Bitrix-структуру проекта.
- Для проверок предпочитай существующие скрипты, Composer и PHPUnit-конфигурацию проекта.
- Не редактируй секреты в `.env` и локальные Bitrix-настройки без явного запроса.
